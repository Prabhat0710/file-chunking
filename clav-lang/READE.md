# Clav Language

Clav is a minimal programming language built from scratch to understand how interpreters and programming languages work internally.

This project focuses on implementing core concepts like tokenization, parsing, and execution in a simple and structured way.

---

## 🎯 Goals

* Build a basic interpreter from the ground up
* Understand how code is transformed and executed
* Keep the design simple and incremental
* Document progress through clear commits

---

## ⚙️ Features (Planned)

* Basic arithmetic operations
* Print statements
* Variables
* Simple expressions

---

## 🧱 Architecture

The interpreter is divided into three main components:

* **Lexer** → Converts source code into tokens
* **Parser** → Builds a structure from tokens
* **Interpreter** → Executes the parsed structure
